# Kassewood Agent – AI Chatbot

A modern, AI-powered chatbot widget built for **Kasselwood Fabricators**. It provides instant support, answers FAQs, and helps visitors get in touch — all through a sleek floating chat interface.

## Features

- 💬 Floating chat widget with smooth open/close animations
- � Live "Online" status indicator
- 🤖 AI-powered responses via n8n + OpenAI
- ❓ Built-in FAQ section
- 💾 Chat history saved per session
- 📱 Fully responsive (mobile + desktop)

## Tech Stack

- **React** + **TypeScript**
- **Vite** (dev server & build)
- **Framer Motion** (animations)
- **n8n** (AI backend workflow)

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/tAlHa-Error404/Kassewood-Agent.git
cd Kassewood-Agent
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
